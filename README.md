# Kimberly McNeil Portfolio

## Description
This is an online portfolio displaying an About Me section, three Projects, and how to Contact me.

## Technologies Used
The technologies used were HTML, CSS, and Font Awesome.

## Website Link
Here is the [Website Link] (https://kimberly-mcneil-portfolio.netlify.app/).

## Challenges and Future Improvements
The Template I used was from a [YouTube Tutorial] (https://www.youtube.com/watch?v=ldwlOzRvYOU) by How to Become a Developer, and the biggest challenge was customizing it to fit my needs. I did not need or want everything that was included, and I wanted to add some of my own touches, such as the icons I used and making it so links took the user to a new tab. Future improvements include adding more projects and experience once I get them, along with perhaps experimenting a bit with color, though I do like the classy and simple feel of the black and white.
